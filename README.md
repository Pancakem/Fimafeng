> A simple HTTP subset web server that supports TLS

### 🏠 [Homepage](https://github.com/Pancakem/Fimafeng)

## Install

```sh
git clone https://github.com/Pancakem/Fimafeng
```

## Usage

```sh
cargo run <host> <port> <path>
```

## Run tests

```sh
cargo test
```
